# Movie Library Management Application
This application has three types of users, each with distinct permissions after logging in.

Both "Angajat" and "Manager" share the following functionalities: sorting movies by type and year, filtering films by type, category, and year, searching for a movie by title, and saving the film database in various formats: JSON, XML, DOC, and CSV. "Angajat" can also perform CRUD operations on films.

"Administrator" is empowered to execute CRUD operations on users, view and filter users by type.
